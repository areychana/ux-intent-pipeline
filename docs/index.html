UX Intent Analysis Pipeline

NLP system that detects user intent and extracts UX themes from qualitative feedback

FastAPI Zero-Shot Classification TF-IDF + KMeans Docker Python 3.10 GSoC 2026

API Endpoints

GET

/health

Liveness probe. Returns API version and status.

POST

/analyze

Full pipeline - intent + topics + summary for a single feedback text.

POST

/intent

Lightweight intent classification only using zero-shot BART model.

POST

/topics

Batch topic extraction using TF-IDF vectorization + KMeans clustering.

POST

/batch

Run full pipeline over multiple feedback items in a single request.
